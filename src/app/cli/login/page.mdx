# dosei login
Learn how to login into your Dosei account using dosei login CLI command.

## dosei login
```bash
dosei login
```
Using this command you can login into your Dosei account using your Github Account.

## Using Dosei Tokens
Alternatively you can use a dosei token generated from the console and set it as an environment variable.
```bash
export DOSEI_TOKEN="you_dosei_token"
```
