# DeployPlex CLI
Learn how to use the DeployPlex command-line interface (CLI) to manage and configure your DeployPlex Projects from the command line.

You can install and configure deployplex using this command:
```bash
pip install -U deployplex
```

## Before you begin

To start using the CLI, you will first need to login into your DeployPlex account.
