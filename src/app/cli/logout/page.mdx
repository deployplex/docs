# dplex logout
Learn how to logout from your DeployPlex account using dplex logout CLI command.

## dplex logout
```bash
dplex logout
```
Using this command you can logout from your DeployPlex account, this will remove your session and remove the stored from
`~/.dplex/credentials.json`
